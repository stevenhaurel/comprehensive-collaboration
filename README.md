# Simple PHP Project

A simple PHP project demonstrating basic PHP functionality including forms, functions, and HTML integration.

## Features

- Display PHP version and server information
- Simple greeting function
- Dynamic content rendering
- Basic form handling
- Clean and responsive design

## Requirements

- PHP 7.0 or higher
- Web server (Apache, Nginx, or PHP built-in server)

## Installation

1. Clone this repository:
```bash
git clone <repository-url>
cd temporary-colaboration
```

2. Run the project using PHP built-in server:
```bash
php -S localhost:8000
```

3. Open your browser and navigate to:
```
http://localhost:8000
```

## Usage

Simply open `index.php` in your web browser. You can:
- View the current PHP version
- See server time
- Enter your name in the form to receive a personalized greeting
